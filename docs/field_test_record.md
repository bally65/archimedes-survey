> 測試方法：使用土壤濕度計（或烘箱法），每個含水量條件下於同一沙坪測試 3 次取平均。

### 1.1 量測方法說明
- **地面承載力**：圓錐貫入試驗（CPT）或簡易穿刺計，單位 kPa
- **推進摩擦力**：用彈簧秤拉動靜止機體，記錄滑動開始瞬間力值（N）
- **螺旋推進效率**：理論推進距離 / 實際前進距離 × 100%（以馬達轉速換算）
