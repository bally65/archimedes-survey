## 2. 電池系統選型

### 2.1 LiFePO4（磷酸鐵鋰）方案（推薦）

相較於 Li-ion 18650，LiFePO4 在戶外潮間帶環境更安全：

| 特性 | LiFePO4 | Li-ion 18650 |
|------|---------|-------------|
| 標稱電壓 | 3.2V/cell | 3.6V/cell |
| 熱失控風險 | 極低（着火溫度 270°C）| 中等（150°C）|
| 循環壽命 | 2000~3000 次 | 300~500 次 |
| 重量能量密度 | 120~160 Wh/kg | 200~260 Wh/kg |
| 耐振動 | 優（鐵鋰硬殼）| 中（需防震）|
| 耐低溫 | 0°C 以上正常 | 需加熱 |
| 成本 | 較高初期 | 較低 |

### 2.2 電池規格設計

```
系統電壓：24V（以 8S LiFePO4 配置）
  8 串：8 × 3.2V = 25.6V 標稱，29.2V 滿充，24.0V 放完

容量需求：
  能耗 120 Wh ÷ 25.6V = 4.7 Ah（100% DoD）
  實際設計（保留 20% 裕量）：4.7 ÷ 0.8 = 5.9 Ah → 選 8 Ah

推薦電池：
  型號：CATL LiFePO4 32700 圓柱電池
  單芯規格：3.2V / 6000mAh（6Ah）
  配置：8S1P（8 顆串聯）
  總能量：8 × 3.2V × 6Ah = 153.6 Wh
  最大放電電流：2C = 12A（滿足 135W / 24V = 5.6A 巡航需求）
  峰值電流：3C = 18A（推進峰值 245W / 24V = 10.2A，僅 56% ≤ 峰值額定）
  重量：8 × 145g = 1.16 kg
  尺寸：Φ32mm × 70mm × 8 顆 → 約 80mm × 80mm × 70mm 排列（2×4）
```

### 2.3 BMS（電池管理系統）規格

```
推薦型號：JBD-SP04S006 （8S LiFePO4 BMS）
或：Daly BMS 8S 24V 20A 帶均衡

功能要求：
  ✅ 過充保護（≥ 3.65V/cell = 29.2V）
  ✅ 過放保護（≤ 2.8V/cell = 22.4V）
  ✅ 過流保護（≥ 20A 即切斷）
  ✅ 短路保護（< 100μs 響應）
  ✅ 溫度保護（> 65°C 充電截止，> 75°C 放電截止）
  ✅ 被動均衡（各芯電壓偏差 < 50mV）
  ✅ 狀態通訊：UART/RS485（可接 RPi4 讀取電量）

尺寸：約 70mm × 50mm × 10mm
```

## 3. 電壓轉換架構

```
電池 25.6V（LiFePO4 8S）
    │
    ├── TB6600 驅動板 × 4（直接 24V 輸入，max 42V）─── NEMA23 × 4
    │
    ├── DC-DC Buck 24V→6V 5A（PCA9685 + 伺服）──────── 手臂 MG996R × 2 + DS3225 × 1
    │     推薦：LM2596S 可調 Buck（NT$35/個）
    │
    ├── DC-DC Buck 24V→5V 5A（RPi4 + 感測器）─────── RPi4 + USB 週邊
    │     推薦：MP1584EN 模組（NT$40/個），需提供穩定 5V/3A
    │
    └── DC-DC Buck 24V→5V 3A（超音波換能器電路）──── MAX9814 + MCP3208

電量計：INA226（I2C 電流/電壓量測 IC，精度 ±0.1%）
  安裝在電池正極串聯 10mΩ 採樣電阻
  → 即時電流、電壓、SOC（State of Charge）→ RPi4 → /battery/voltage ROS2 topic
```

## 4. SOC 電量計算與 RTH 觸發

```python
# battery_monitor.py（安裝於 /ros2/archimedes_survey/）
# INA226 → I2C → RPi4 → ROS2 /battery 話題

BATT_FULL_V  = 29.2   # 8S LiFePO4 滿充
BATT_EMPTY_V = 22.4   # 8S LiFePO4 放完（保護截止）
BATT_RTH_V   = 23.6   # 觸發 RTH 的電壓（≈ 20% SOC）
BATT_WARN_V  = 24.8   # 低電量警告（≈ 30% SOC）

def voltage_to_soc(v: float) -> float:
    """LiFePO4 放電曲線近似（平台區較平坦）。"""
    if v >= BATT_FULL_V:  return 100.0
    if v <= BATT_EMPTY_V: return 0.0
    # 線性近似（LiFePO4 平台區為主）
    return (v - BATT_EMPTY_V) / (BATT_FULL_V - BATT_EMPTY_V) * 100.0
```

### 分級降功率策略

| SOC | 電壓 | 動作 |
|-----|------|------|
| 100~50% | > 26.0V | 正常作業 |
| 50~30% | 24.8~26.0V | ⚠️ 警告 + 限速至 70% |
| 30~20% | 23.6~24.8V | 🔶 強制返航（RTH），禁止新任務 |
| <20% | < 23.6V | 🚨 緊急降速至 50%，最短路徑 RTH |
| <10% | < 22.8V | 發 LoRa 求救 + 停車等待 |

## 5. 充電系統

```
充電器規格：
  輸入：100-240V AC，50/60Hz
  輸出：29.2V（8S LiFePO4 滿充電壓），最大充電電流 5A
  充電時間：8Ah ÷ 5A = 1.6h（80% → 100%），全程約 2h
  推薦型號：iCharger X8（多功能平衡充，支援 LiFePO4 8S）
  或：自製 CCCV 充電板（LM338 + 精密稳压）

充電介面：
  艙外 XT60 防水公頭（面板安裝型，IP67）
  充電時勿浸水

攜帶式充電（田野）：
  100W 太陽能板 + MPPT 控制器（24V 系統）
  晴天 4~5h 充滿（800W·h÷100W=8h，實際 MPPT 效率 75%→5.3h）
```

## 7. 安全注意事項

```
⚠️  LiFePO4 電池雖比 Li-ion 安全，但仍需注意：
  - 禁止過充（超過 29.2V）或過放（低於 22.4V）
  - 不得在高溫環境（> 45°C）充電
  - 電池艙需設計通氣孔（或採用通氣閥）
  - 攜帶或運輸時套裝防靜電袋
  - 發現電芯鼓脹 → 立即停止使用，隔離處理

🌊  潮間帶作業額外注意：
  - 電池艙須通過 IP68 測試方可入水
  - 每次任務結束後檢查電池艙內濕度（SHT31）
  - 若密封艙內濕度 > 70% RH → 立即開艙乾燥
```
