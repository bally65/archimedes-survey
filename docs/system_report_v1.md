## 01 系統概覽

**設計目標：** 在沙灘環境中自動巡線，偵測奧螻蛄蝦（Lysiosquillina）洞口，並以探針取樣記錄。

**架構：**
```
archimedes-survey/
├── mechanics/        # 螺旋推進計算、風力分析
├── arm/              # 機械手臂運動學
├── sensing/          # 洞口影像辨識
├── simulation/       # 沙土+水文模擬
├── navigation/       # 自動巡線路徑規劃
└── visualization/    # 族群密度地圖輸出
```

## 02 環境分析

**土壤力學（現場數據校準）**
- 現場測試：49.7kg 體重踩出 5cm 深度
- 腳底接觸面積推算腳壓力 ≈ 1,393 Pa
- 承載力（乾沙 10% 濕度）：~14 kPa
- 承載力（濕沙 90% 濕度）：~1.4 kPa（近液態）
- 最難作業濕度：40~60%（半硬沙，插針力最大）

**風力條件**
- 最大設計風速：40 m/s（颱風級）
- 颱風風力：>130 N
- 濕沙摩擦力：~70 N（機體重 22kg）
- **風力 > 摩擦力，需要錨定機制**

**錨定策略：**
1. 低風阻收納姿態（手臂/感測器全收）
2. 螺旋反轉「咬進沙裡」錨定
3. 作業風速限制 < 20 m/s

## 03 螺旋推進

**幾何設計（確定參數）**
- 管材：6 吋 PVC 管（外徑 168mm）
- 螺旋外徑：14cm（依腳壓校準）
- 每段：36°，10 片拼一圈
- 葉片寬：50mm，厚：5mm
- 材料：PETG 列印
- 每條螺旋：5 圈 = 50 片
- 雙螺旋（A/B 各偏移 180°）= 共 100 片

**列印資訊**
- STL 檔：`blade_A_seg01~10`, `blade_B_seg01~10`, `blade_FULL_PREVIEW_1turn`
- 每批 4 片（220mm 熱床 2×2 排列）
- 共 25 批 × 2 小時 = 約 50 小時總列印時間

**馬達選型**
- 推進馬達：依土壤阻力計算（待補完整規格）

## 06 電控系統

**電源架構**
- 電池容量：480 Wh
- 系統功耗：~120 W
- 實際續航：**3 小時以上**
- 電池電壓：24V（馬達）+ 5V/3.3V 降壓（控制板）

**主控架構（建議）**
- 高層邏輯：Raspberry Pi（Python / 影像辨識）
- 即時馬達控制：STM32 MCU

**防水等級：** 沙灘環境，需 IP65 以上

## 07 風險評估

| # | 風險 | 應對策略 |
|---|------|----------|
| 1 | 風力超過摩擦力 | 螺旋錨定 + 作業風速限制 |
| 2 | 高濕度沙土（近液態） | 螺旋推進天然適合，效率反升 |
| 3 | 中濕度半硬沙插針阻力大 | 馬達扭矩留足餘裕 |
| 4 | 機身密度高，重心偏移 | 手臂受力計算先行，限制感測器重量 |
| 5 | 感測器防水防塵 | IP65 封裝，沙灘測試驗證 |
| 6 | 電池在高溫沙灘膨脹 | LiFePO4（熱穩定性佳）優先 |
