# YOLOv8 訓練影像蒐集策略
**美食奧螻蛄蝦（Austinogebia edulis）洞口影像採集與標注指南**
版本：1.0  日期：2026-03-17

---

## 1. 目標蒐集規模

| 類別 | 建議最低數量 | 理想數量 | 說明 |
|------|------------|---------|------|
| 洞口（burrow） | 200 張 | 500+ | 主要偵測目標 |
| 背景（無洞口泥灘）| 100 張 | 200+ | 降低假陽性 |
| 困難樣本（螃蟹洞、礫石）| 50 張 | 100+ | 提升判別力 |
| **合計** | **350 張** | **800+** | |

YOLOv8-nano 最低有效訓練資料量約 200 張，但彰化泥灘地條件特殊（高反光、潮水干擾），建議至少 400 張。

## 2. 最佳採集時機

### 2.1 季節選擇

| 月份 | 洞口可見度 | 備注 |
|------|----------|------|
| 11月~翌年3月 | ⭐⭐⭐⭐⭐ 最佳 | 交配季前後，洞口密度最高 |
| 4月~5月 | ⭐⭐⭐ 良好 | 孵卵期，母蝦留巢，洞口仍清晰 |
| 6月~10月 | ⭐⭐ 普通 | 颱風、高溫干擾，洞口可能被沖平 |

**首批採集建議時間：11月~翌年2月（冬末春初）**

### 2.2 潮汐時機

```
最佳採集窗口：退潮後 1~2 小時
  - 泥灘剛露出，洞口輪廓最清晰（有水分痕跡）
  - 進水口凹陷明顯，出水口火山口突起顯著
  - 避免完全乾涸（泥面反光過強，對比度下降）

次佳：低潮時刻前後 30 分鐘
  - 水分仍留在洞口周圍，形成視覺對比

禁止採集：
  - 漲潮期間（洞口被水覆蓋）
  - 剛下大雨後（泥面被沖平，洞口模糊）
```

## 4. 拍攝技術規範

### 4.1 相機設定

```
手機推薦：iPhone 15+ 或 Samsung S24+（1200萬像素以上）
相機推薦：可防潑水的 APS-C 相機 + 標準鏡頭

解析度：至少 1920×1080，建議 3840×2160（4K）
格式：JPG（YOLOv8 訓練標準格式）
      RAW + JPG（後備，可調色溫補正高反光）
白平衡：自動或「陰天」（泥灘地反光多為白光）
曝光補償：-0.3 EV（防止過曝損失洞口細節）
```

### 4.2 拍攝角度與距離（對應機器人視角）

機器人相機安裝位置：平台頂面，離地高度約 156mm + 相機座高 ≈ 180mm

```
俯角（Depression angle）：30°~60°（正常巡航速度時相機前傾 45°最多）
拍攝距離（地面到相機）：15cm ~ 50cm

→ 建議採集角度：
   方案 A（主要）：手持相機，距地面 20~40cm，俯角 45°，模擬機器人視角
   方案 B（補充）：正上方俯視（90°），捕捉洞口全形（進/出水口特徵）
   方案 C（環境）：遠距 1~2m 俯角 30°，涵蓋背景紋理（避免過擬合）
```

### 4.3 多樣性要求

每個地點至少包含以下變化：

```
光線條件：
  [ ] 正午強光（泥面高反光）
  [ ] 陰天漫射光（最佳！均勻對比）
  [ ] 清晨側光（洞口陰影深，最易辨識）

泥灘狀態：
  [ ] 剛退潮（濕潤，顏色深）
  [ ] 退潮 1h 後（表面半乾，顏色較淺）
  [ ] 有淺水窪（10mm 水深，洞口在水中）

洞口狀態：
  [ ] 單一洞口（進水口）特寫
  [ ] 單一洞口（出水口）特寫（火山突起形態）
  [ ] 一組雙洞口（進+出，距離 21~26cm）
  [ ] 有蝦猴爬出的洞口（如果幸運拍到）

干擾物（作為困難樣本）：
  [ ] 招潮蟹洞（圓形，較小，直徑 1~2cm）
  [ ] 貝殼堆積
  [ ] 礫石
  [ ] 水流痕跡（類似洞口圓形）
```

## 5. 標注工具與格式

### 5.1 推薦工具

```
Roboflow（推薦）：
  - 網頁版：https://roboflow.com（免費方案支援 1000 張）
  - 支援直接輸出 YOLOv8 格式
  - 自動資料擴增（旋轉、翻轉、亮度調整）
  - 可多人協作標注

LabelImg（本地）：
  - 安裝：pip install labelImg
  - 使用：labelImg images/ labels/
  - 輸出：YOLO txt 格式

CVAT（開源，進階）：
  - 支援多邊形、關鍵點標注
  - 適合後期要做語意分割升級
```

### 5.2 標注規則

```
類別定義：
  0: burrow_inlet   進水口（凹陷，較大）
  1: burrow_outlet  出水口（突起，較小）
  2: burrow_pair    成對雙洞口（標注包含兩個洞口的矩形框）

標注邊界框規則：
  - 邊界框應緊貼洞口邊緣，外留 5px 緩衝
  - 若洞口部分被水或泥覆蓋，可見部分 > 50% 才標注
  - 不確定是否為蝦猴洞的樣本 → 跳過（寧缺毋濫）

資料夾結構（YOLOv8 標準）：
  datasets/burrow/
    images/
      train/   (70%)
      val/     (20%)
      test/    (10%)
    labels/
      train/
      val/
      test/
```

## 7. 資料擴增策略（Augmentation）

在 `vision/train_burrow.py` 中已有基本設定，建議加入：

```python
# 潮間帶特定擴增
augmentation_config = {
    # 光線模擬
    "brightness": (-0.3, +0.2),    # 強光 → 陰天
    "contrast":   (0.7, 1.4),
    "hue_saturation": (-15, +15),  # 泥灘反光色偏
    "blur":       (0, 1.5),        # 相機震動模糊

    # 幾何擴增（機器人移動視角）
    "rotate":     (-30, +30),      # 機器人轉彎
    "flip_h":     True,            # 左右對稱
    "flip_v":     False,           # 不上下翻（洞口有方向性）
    "scale":      (0.7, 1.3),      # 不同拍攝距離

    # 潮間帶特殊效果
    "noise":      0.02,            # 模擬泥水濺射
    "fog":        0.1,             # 模擬漲潮時水霧
    "cutout":     2,               # 部分遮擋（草、貝殼）
}
```
