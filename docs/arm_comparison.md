# 機械手臂方案比較 — Archimedes-Survey v2
> 需求：3-DOF，探針插沙，承重 1–2 kg，臂重 < 3 kg，預算 < NT$5,000，可防水

---

## 需求摘要

| 指標 | 目標值 |
|------|--------|
| 自由度 | 3 DOF（垂直平面：肩仰俯 + 肘伸縮 + 腕旋） |
| 有效承重 | 1–2 kg（探針 + 感測器 ~200g，安全係數 5x） |
| 最大伸展距離 | 30–50 cm |
| 手臂自重 | < 3 kg |
| 預算 | < NT$5,000（< USD $150） |
| 結構 | 可 3D 列印 / 現成零件 |
| 防水 | IP65（沙灘鹽水環境） |
| ROS 相容 | 加分項，非必要 |

## 各方案詳細分析

### 1. SO-ARM100（HuggingFace / LeRobot）⭐ 次選

- **GitHub：** https://github.com/huggingface/lerobot
- **馬達：** Feetech STS3215 串列總線伺服（~USD $12–15 每顆）
- **特點：** 完全可 3D 列印、社群活躍（HuggingFace 生態）、Python-native SDK
- **限制：**
  - 0.5 kg 承重對 1–2 kg 目標偏低（需換強力伺服）
  - 伺服馬達無防水，需額外封裝
  - 若升級伺服至 DS3218 20 kg·cm，承重可提升至 ~1.2 kg
- **防水改裝方案：** 3D 列印密封伺服殼 + O-ring 溝槽 + 矽膠封膠 + 線纜穿矽膠束線孔
- **適合：** 預算緊且有軟體開發能力的情況

---

### 2. 自製 3DOF 沙灘臂（本專案推薦方案）⭐⭐ 首選

**設計概念：** 全垂直平面 3 關節（最符合「插針入沙」動作）

```
  [平台] → J1（底座左右旋 ±60°）
          → J2（肩：上仰 0–90°）
          → J3（肘：前伸 0–120°）
          → [探針]
```

**伺服選型（強力版）：**

| 伺服型號 | 扭矩 | 電壓 | 單價 (NT$) | 用途 |
|---------|------|------|-----------|------|
| DS3218MG 20 kg·cm | 20 kg·cm (1.96 N·m) | 6.8V | ~350 | J2 肩關節（最大力矩） |
| MG996R 13 kg·cm | 13 kg·cm (1.27 N·m) | 6V | ~180 | J3 肘關節 |
| MG995 13 kg·cm | 13 kg·cm (1.27 N·m) | 6V | ~150 | J1 底座旋轉 |
| 伺服電源降壓模組 24V→6.8V | — | — | ~120 | 從主電池供電 |

**費用估算：**

| 品項 | 費用 (NT$) |
|------|-----------|
| 伺服馬達 × 3 | ~680 |
| PETG 列印結構件（~300g）| ~180 |
| 伺服控制板（PCA9685 I2C）| ~120 |
| 鋁合金連桿 + 螺絲 | ~300 |
| 防水殼 + 矽膠 + O-ring | ~400 |
| **合計** | **~NT$1,680–2,200** |

**優勢：**
1. 最便宜（NT$1,680–2,200，遠低於 NT$5,000 預算）
2. 專為本機設計，可直接整合到平台架構
3. IP65 防水最容易實現（自己設計密封殼）
4. 3 DOF 足夠完成「肩仰→肘伸→探針入沙」動作
5. PCA9685 透過 I2C 接 RPi，程式簡單

**限制：**
- 無現成 URDF，需手動建 ROS2 模型（工程量 ~2–4 小時）
- 需要 CAD 設計能力（Fusion 360 / Onshape）

---

### 3. OpenManipulator-X（ROBOTIS）— 技術標竿，預算超標

- **優點：** ROS2 完整支援（MoveIt 2、Gazebo 模擬）、Dynamixel 品質高
- **缺點：** ~NT$19,000–29,000，超出預算 4–6 倍；Dynamixel 無防水
- **結論：** 作為參考設計，但不採購

---

### 4. BCN3D Moveo / AR4 / Thor — 皆不適合

- 臂重 3–6 kg（超出本機載重設計）
- 費用 NT$10,000–30,000（超預算）
- 適合固定式桌面機器人，非沙灘移動平台

---

## 最終推薦

### Phase 2 採用方案：自製 3DOF + SO-ARM100 參考設計

| 決策 | 內容 |
|------|------|
| 結構設計 | 參考 SO-ARM100 連桿比例，自行設計適合本機的 3-DOF 臂 |
| 伺服馬達 | J1: MG995, J2: DS3218MG, J3: MG996R |
| 控制 | PCA9685（I2C）→ Raspberry Pi 4，Python 驅動 |
| 防水 | 自製 PETG 密封殼 + O-ring，目標 IP65 |
| 費用 | ~NT$1,680–2,200（大幅低於預算） |
| ROS2 整合 | 手動建 URDF + joint_state_publisher，預估 2–4 h 工程量 |

## 手臂 CAD 設計下一步

1. **確定探針重量**（估計 ~100–200 g）→ 校驗 DS3218MG 扭矩餘裕
2. 在 Fusion 360 / Onshape 設計：
   - J1 底座旋轉座（固定到機器主平台）
   - J2 肩關節連桿（長度 ~20 cm）
   - J3 肘關節連桿（長度 ~20 cm）
   - 探針夾持座
3. 使用 PETG 列印，所有伺服封裝加 O-ring 溝槽
4. 建立 ROS2 URDF 模型（可用 SW2URDF 插件）
