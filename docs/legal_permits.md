## 1. 法規框架總覽

在彰化縣伸港、王功等保育區使用自主機器人進行調查，涉及以下法規：

| 法規 | 主管機關 | 關鍵條文 | 對本計畫的影響 |
|------|---------|---------|--------------|
| **漁業法** | 農業部漁業署 | §44（繁殖保育區設立），§48（禁止採捕行為）| 保育區內需申請研究許可 |
| **水域遊憩活動管理辦法** | 交通部觀光署 | §8（非機動載具水域活動申請）| 需確認機器人是否屬「水域遊憩活動」|
| **野生動物保育法** | 農業部生物多樣性研究所 | §17（干擾保育類動物）| 若附近有珍貴稀有鳥類棲地需額外申請 |
| **海岸管理法** | 內政部國土管理署 | §23（一般管制區禁止行為）| 作業區域分類影響許可難度 |
| **科學技術基本法** | 國科會 | 研究豁免條款 | 學術研究可申請免除部分限制 |

## 2. 彰化螻蛄蝦保育區規定

### 2.1 現有保育區

根據農業部漁業署公告，彰化縣設立三處螻蛄蝦繁殖保育區：

```
(1) 伸港繁殖保育區
    範圍：伸港鄉沿岸潮間帶（詳細座標見漁業署公告）
    禁止行為：採捕螻蛄蝦（含蝦猴）及其幼體
    主管機關：彰化區漁會 / 農業部漁業署第二區漁業管理處

(2) 伸港（二）繁殖保育區
    範圍：伸港（一）向北延伸區域
    規定：同上

(3) 王功繁殖保育區
    範圍：王功漁港周邊潮間帶
    特殊規定：保育班員定期進行高壓水柱翻土復育作業
    主管機關：同上
```

### 2.2 保育區的機器人作業合法性分析

```
✅ 允許的行為：
  - 非接觸式遙測（超音波掃描、拍照、GPS 記錄）
  - 水上移動（不破壞底質）
  - 學術研究調查（需申請許可）

⚠️  灰色地帶（需事先確認）：
  - 超音波波束照射泥灘地（不屬於「採捕」，但需確認）
  - 機器人螺旋葉片與泥面接觸（可能被認定「攪動底質」）

❌ 明確禁止：
  - 採集蝦猴個體（即使是研究目的，未有許可不得為之）
  - 挖掘或破壞洞穴
  - 在繁殖期（通常 3~5月）進行任何可能干擾繁殖的作業
```

## 3. 許可申請流程

### 3.1 學術研究許可（推薦路徑）

**路徑 A：透過學術機構申請（最快）**

```
申請機構：國立中興大學（NCHU）或逢甲大學生物科技系
          農業部水產試驗所（TFRIN）海水繁殖研究中心

流程：
  1. 聯絡學術單位，說明研究目的與機器人技術
     → 目標：與相關教授簽署合作研究協議

  2. 由學術單位以「海洋生態調查研究計畫」向農業部漁業署申請
     → 文件：研究計畫書、機器人技術說明、不影響保育聲明

  3. 農業部漁業署審核（通常 2~4 週）
     → 可能附帶條件：指定作業時段、研究人員陪同、資料共享義務

  4. 取得研究許可後，通知彰化區漁會，作業前 3 天知會保育班
```

**路徑 B：直接向彰化區漁會申請（地方路徑）**

```
聯絡資訊：
  彰化區漁會
  地址：彰化縣鹿港鎮草港里草港 10 號
  電話：04-7784106
  網址：http://www.chafaa.org.tw

申請文件：
  - 申請書（說明目的、日期、地點、設備）
  - 機器人技術說明（無採捕功能聲明）
  - 研究成果資料共享承諾書
  - 保險證明（第三人責任險，建議保額 100 萬/次）

預計處理時間：2~3 週
```

## 4. 無人載具相關法規

### 4.1 水上無人載具（USV）

目前台灣無專法規範水面自主機器人（USV），適用：

```
水域遊憩活動管理辦法（交通部觀光署）：
  - 適用範圍：供遊憩使用的水域活動
  - 研究用機器人：可能適用「非遊憩目的」豁免
  - 建議：主動向觀光署諮詢，取得不適用書面確認

海商法、船舶法：
  - 長度 < 2m、排水量 < 1 噸的小型載具通常不適用船舶法
  - Archimedes 尺寸遠小於法規門檻，無需船舶檢驗
```

### 4.2 無線電（LoRa 433MHz）

```
LoRa 433MHz 在台灣：
  - 433MHz ISM 頻段（426~438MHz）：不需申請，限制最大輸出功率 10mW（EIRP）
  - Ra-02 模組預設輸出功率：100mW → 需調降至 10mW 或取得業餘無線電執照
  - 建議：申請業餘無線電執照（丙類以上），可合法使用 433MHz @ 50W

  或改用 915MHz 頻段：台灣較少使用（需另確認 NCC 規範）
```
