# 防水與 IP 等級設計規範
**Archimedes Survey Robot — 潮間帶作業密封方案**
版本：1.0  日期：2026-03-17

---

## 1. 作業環境需求分析

| 環境參數 | 彰化潮間帶實際條件 | 設計要求 |
|----------|-------------------|----------|
| 鹽度 | 海水（35‰）潮間帶混合水 | 防鹽霧腐蝕，IP68 |
| 浸水深度 | 最深退潮作業約 5cm 水深 | IP68（1m/30min 以上）|
| 浸水時間 | 最長連續 8.5h 作業 | 持續浸潤 + 毛細現象防護 |
| 泥沙侵入 | 細粒黏土（粒徑 9Φ）+ 粉砂 | 防塵 IP6X（完全密閉）|
| 溫度 | 夏季 35°C 外部，密封艙內可達 55°C | 熱管理（見第 4 節）|
| 紫外線 | 台灣夏季高 UV | 外殼材質 UV 穩定 PC 或 ABS+UV 穩定劑 |
| 衝擊 | 螺旋推進接觸礁石、碎貝 | 外殼至少 2mm 壁厚 HDPE/PETG |

**目標 IP 等級：IP68（全面密閉，可承受 1m 水深 30 分鐘以上持續浸水）**

## 2. 電子密封艙設計（主控艙）

### 2.1 艙體規格

```
主控艙尺寸（建議）：
  外部：220mm × 130mm × 80mm（可容納 RPi4 + 電池管理板 + LoRa）
  壁厚：3mm HDPE（高密度聚乙烯，耐鹽水腐蝕，可 FDM 列印）
  蓋板：透明 PC（聚碳酸酯）+ O-ring 槽

密封方式：
  雙 O-ring 壓縮密封（EPDM 橡膠，耐鹽水）
  溝槽規格：AS568-234 O-ring（23mm ID × 3.5mm 截面）
  壓縮量：20~25%（最佳密封壓縮比）
  螺栓：M3 × 8 不鏽鋼（SUS316），間距 ≤ 40mm
```

### 2.2 線材穿越（Cable Gland）

每條穿越線材使用獨立的 IP68 防水接頭：

| 線材 | 推薦型號 | 線徑 | 數量 |
|------|----------|------|------|
| 電源輸入（24V XT60） | M16 Cable Gland（IP68）| 7-10mm | 1 |
| 馬達訊號（TB6600 step/dir，x4） | M12 Cable Gland 4孔 | 5-7mm | 2 |
| GPS 天線延長線 | M8 Cable Gland（SMA） | 4mm | 1 |
| IMU / ADS1115 I2C | M12 Cable Gland 4孔 | 5mm | 1 |
| 相機 USB | M16 Cable Gland | 8mm | 1 |
| LoRa 天線 SMA | M8 SMA 防水穿牆頭 | — | 1 |
| 超音波換能器（同軸）| M12 BNC 防水穿牆 | 7mm | 1 |

**安裝原則：**
- 所有 Cable Gland 安裝在艙體**側面**或**底部**，避免安裝於頂蓋（防止水從上方積累）
- 安裝前塗抹 Loctite 5900 矽利康密封劑於螺紋
- 線材穿越後進行 1m/30min 浸水測試

### 2.3 壓力補償閥（Pressure Relief Valve）

密封艙在溫差變化下內外壓力差可達 ±30 kPa，長期壓力差會破壞 O-ring：

```
推薦元件：M12 Gore-Tex 膜片透氣防水通氣閥（Vent Plug）
型號：GORE® Protective Vents - M12 × 1.5
特性：防水 IPX8，允許氣體雙向通過，阻止液態水侵入
安裝位置：艙體頂面（遠離浸水方向）
數量：1 個
```

## 3. 馬達與傳動系統防水

### 3.1 NEMA23 步進馬達防水處理

原廠 NEMA23 的 IP 等級通常僅 IP40，需額外處理：

```
措施 1：電機軸端密封
  - 軸端加裝唇型密封圈（Oil Seal），型號 TC 14×25×7（14mm 軸徑）
  - 前後軸各一個，安裝在端蓋與管體配合面

措施 2：接線口灌封
  - 馬達接線盒用環氧樹脂灌封（Epoxy potting）
  - 使用 3M Scotchcast 40 灌封膠，耐溫 -55°C ~ 130°C

措施 3：整體護套
  - 馬達外套熱縮管（2:1 壁厚，直徑 90mm）+ 矽利康密封兩端
  - 作業後以淡水沖洗馬達表面
```

### 3.2 軸承（6205 滾珠軸承）防水

```
替換為：6205-2RS（雙橡膠密封型，IP64 等級）
潤滑油脂：換用 Mobilgrease XHP222（耐鹽水脂，不被海水乳化）
更換週期：每 50 小時作業或每 3 個月
```

### 3.3 螺旋葉片防腐

```
材質：PETG（耐鹽水優於 PLA）
後處理：
  1. 乙酸乙酯蒸氣平滑處理（消除層紋毛細縫）
  2. 噴塗 Rust-Oleum 防銹噴漆（底漆 + 面漆各一道）
  3. 或換 HDPE 機加工葉片（長期使用建議）
```

## 4. 熱管理設計（密封艙散熱）

密封後無法自然對流散熱，RPi4 在 5V/3A 全載下熱耗約 15W：

### 4.1 被動散熱方案（推薦）

```
方案：銅導熱貼 → 艙壁導熱 → 外部散熱片

元件：
  - RPi4 CPU 銅導熱貼片（3mm 厚，導熱係數 6 W/m·K）
  - 導熱墊（Bergquist GP3000，厚 3mm）
  - 外部鋁散熱片（80mm×80mm×15mm，黏貼於艙體頂面外側）
  - 艙體壁厚局部減薄至 1mm（散熱區域）

預估熱阻：
  CPU → 艙壁 → 外部散熱片：總熱阻約 5~8 °C/W
  15W × 6.5 °C/W = +97°C 溫升 ← 過高！
```

### 4.2 主動散熱方案（建議實施）

```
方案：微型潛水泵水冷循環

元件：
  - 5V 微型潛水泵（JT-180）：流量 180L/h，密封型
  - 銅水冷塊（40mm×40mm，RPi4 CPU 貼合）
  - 軟管（5mm ID 矽膠管）導出艙外，用海水作為冷卻介質
  - 外艙管路：螺旋銅管（表面積 200cm²）散熱至環境水

優點：海水 18~25°C 是天然冷卻劑，熱容量極大
缺點：管路穿艙需額外 Cable Gland，增加密封點
```

### 4.3 溫度監控

```
感測器：MCP9808（I2C，已在 BoM 中）安裝於密封艙內
觸發邏輯：
  if T_core > 75°C:   降頻至 1200 MHz（RPi4 軟限制）
  if T_core > 85°C:   強制 RTH + 停止高耗電節點
  if T_core > 90°C:   緊急斷電（GPIO 繼電器切斷）
```

## 5. 手臂與超音波換能器防水

### 5.1 3-DOF 手臂伺服

```
MG996R 原廠 IP 等級：無評級（IP20）

處理方式：
  1. 伺服機接線用熱縮管 + 矽利康密封
  2. 伺服機外殼塗抹 Conformal Coating（防濕塗料，如 MG Chemicals 419D）
  3. 手臂連桿設計防水槽，避免泥水積在轉動部
  4. 定期（每次任務後）以淡水沖洗 + 壓縮空氣吹乾
```

### 5.2 超音波換能器（PBZT-200）

```
換能器正面：壓電陶瓷 + 鑄聚氨酯（PU）聲學匹配層
  - PU 聲阻抗約 2.5 MRayl，與泥沙聲阻抗接近 → 良好耦合
  - 原廠防水等級：IP68（浸水 10m）

同軸電纜：
  - 使用 RG-174 同軸電纜（外徑 2.8mm，耐彎曲）
  - 電纜與艙體穿越點用 M12 BNC 防水穿牆頭
  - 外部裸露段套 FEP 熱縮管（耐鹽水腐蝕）

安裝位置：
  - 手臂末端探針管外側，與探針中心軸成 15° 角（洞口量測最佳角）
  - 固定夾使用 PP（聚丙烯）材質，不鏽鋼螺絲
```

## 6. 整車防水測試程序

### 出廠前測試（每次組裝後）

```
Step 1 ─ 目視檢查
  [ ] 所有 Cable Gland 旋緊（手緊 + 1/4 圈工具）
  [ ] O-ring 無損傷、無扭曲
  [ ] 螺栓全部鎖緊（扭力 0.6 N·m for M3）

Step 2 ─ 氣壓測試（0.3 bar 正壓，10 分鐘）
  [ ] 用自行車打氣筒向艙內充氣至 0.3 bar（約 3 psi）
  [ ] 塗抹肥皂水於所有密封面，觀察是否起泡
  [ ] 若無洩漏：通過

Step 3 ─ 浸水測試（1m/30min）
  [ ] 整車浸入裝滿淡水的容器
  [ ] 30 分鐘後取出，內部無滲水

Step 4 ─ 功能測試（泡水後）
  [ ] RPi4 正常開機
  [ ] 所有感測器回傳正常數值
  [ ] 馬達驅動正常
```

### 任務前現場快速確認（每次出發前）

```
[ ] 外殼目視無裂縫
[ ] Cable Gland 無鬆動
[ ] 通氣閥未堵塞
[ ] 密封艙內濕度讀數 < 60% RH（MCP9808 旁加裝 SHT31 濕度感測）
```
